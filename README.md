# Token Animator

Мощный конвертер MP4 в WebM/GIF с круговой маской, каталогом рамок и ping-pong анимацией для VTT токенов.
Работает полностью **офлайн** после первоначальной настройки.

---

## Запуск

Дважды кликни **`START_RUN_STROY.bat`** (этот файл работает из любой папки и на любом ПК) — всё остальное происходит автоматически:

1. Если папка `Binaries\lib\` пустая — скачивает библиотеки FFmpeg (~31 МБ, только при самом первом запуске).
2. Запускает локальный многопоточный HTTP сервер.
3. Открывает браузер на `http://localhost:3000`.

---

## Использование

1. Перетащи видео в окно.
2. Открой **«Каталог рамок»**, чтобы выбрать нужную рамку (ты можешь закидывать свои рамки в папку `Frames/`).
3. Двигай рамку мышью и растягивай её за уголки, чтобы идеально подогнать под видео.
4. **Обрезка видео** — встроенный инструмент для выбора нужного куска видео с защитой от слишком коротких фрагментов (минимум 3 секунды).
5. **Ping-pong** — бесшовная петля (оригинал + реверс).
6. Выбери формат (WebM или GIF) и нажми **«Создать»** — файл скачается автоматически!

## Структура проекта

```text
FVTT_Actor_animation\
├── START_RUN_STROY.bat  — Главный файл запуска (портативный скрипт)
├── LICENSE.md           — Лицензия на использование (CC BY-NC 4.0)
├── README.md            — Эта инструкция
├── TEST\                — Папка, куда сохраняются промежуточные видео
├── Frames\              — Папка с твоими рамками (кидай сюда свои картинки)
└── Binaries\            — Системные файлы (исходный код)
    ├── app.js           — Логика интерфейса
    ├── index.html       — Верстка
    ├── style.css        — Стили
    ├── server.py        — Быстрый локальный Python сервер
    └── lib\             — Ядро FFmpeg (создаётся автоматически)
```
