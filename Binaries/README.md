# Token Animator

Конвертер MP4 в WebM с круговой маской и ping-pong анимацией для VTT токенов.
Работает полностью **офлайн** после первоначальной настройки.

---

## Запуск

Дважды кликни **`START_RUN_STROY.bat`** (или красивый ярлык `START_RUN_STROY`) — всё остальное происходит автоматически:

1. Если `lib\` папка пустая — скачивает FFmpeg (~31 МБ, только первый раз)
2. Запускает локальный HTTP сервер
3. Открывает браузер на `http://localhost:3000`

---

## Использование

1. Перетащи MP4 в окно
2. Двигай рамку мышью — позиционируй персонажа
3. Ползунок — масштаб кадра
4. **Круглая маска** — прозрачность за пределами круга
5. **Ping-pong** — бесшовная петля (оригинал + реверс)
6. Выбери разрешение токена
7. Нажми **«Создать WebM»** — файл скачается автоматически

## Структура

```
FVTT_Actor_animation\
├── START_RUN_STROY.bat            — Запуск (всё в одном)
├── index.html
├── style.css
├── app.js
├── server.js            — Node.js fallback сервер
├── download_libs.bat    — Скачать FFmpeg отдельно (опционально)
└── lib\                 — FFmpeg (создаётся автоматически)
    ├── ffmpeg.js
    ├── 814.ffmpeg.js
    ├── ffmpeg-util.js
    ├── ffmpeg-core.js
    └── ffmpeg-core.wasm
```
